Description
----------

Event based irc client library.


Features
---------
* Event based
* Automatic Reconnect
* Handles basic irc demands for you (PING, VERSION)

Install
----------
	$ git clone git@github.com:thoj/Go-IRC-Client-Library.git
	$ cd Go-IRC-Client-Library
	$ make 
	$ make install
